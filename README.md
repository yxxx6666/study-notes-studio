# Study Notes Studio 学霸学习笔记

版本：v1.4.1

把课堂、网课、读书、讲座、文章和复习材料整理成清爽、有重点、能复习的 3:4 视觉学习笔记。

## 使用方法

在 Codex 中直接发送：

```text
/imagegen $study-notes-studio 把下面内容整理成学习笔记，3:4 比例。
（原文正文直接粘在这里）
```

说明：
- `/imagegen` 强制使用图片生成模型
- `$study-notes-studio` 调用本 Skill
- 页数由 Skill 根据内容自动判断，无需手动指定
- 生成后会自动检查比例，不符合 3:4 会自动重试

当前版本：v1.4.1
## v1.4.1 核心升级

- 出图默认且强制使用图片生成模型，禁止以中文可读或严格比例为由擅自切到 Pillow / HTML。
- 所有模式出图前先做内容前置理解：识别类型、抽取计数、决定逐项列全还是合并大类。
- 标题数字必须等于画面项数；合并大类要诚实标注映射（如 12 个习惯→4 大类）。
- 页码与内部约束（“不要写成……”等）不得画进图片，只用于文件名或交付说明。

## v1.4.0 核心升级

- 保留总览笔记、长文配图、完整笔记三种模式路由。
- 长文配图不默认承诺完整覆盖。
- 用户要求完整、不遗漏、覆盖全文时，自动进入完整笔记模式。
- 完整笔记必须建立知识点编号清单和页码—知识点覆盖矩阵。
- 每页新增一个完全原创的“学习锚点图案”。
- 锚点只使用普通学习物件、路径符号、记忆符号或抽象结构符号，不使用角色 IP。

## 默认视觉

全画布学习笔记页。整张图片画布本身就是笔记页，不生成纸张边缘、周围背景、投影、笔记本书脊、桌面背景、强透视或实拍样机。

默认严格 3:4 竖版。

## 安装

把整个 `study-notes-studio` 文件夹放入 Codex skills 目录：

```text
C:\Users\<用户名>\.codex\skills\study-notes-studio\
```


## v1.4.0 清洁审核

安装包内公开文档与配置文件已清理外部参考命名，只保留原创学习锚点图案体系。


## v1.4.0 执行修复

- 用户指定图片模型时，继续使用图片模型。
- 用户禁止 HTML 时，不使用 HTML。
- 3:4 图片必须生成后检查比例。
- 8 个以上编号项默认拆页。


## v1.4.0 视觉修复

- 默认更像手写学生复习笔记。
- 避免信息图、海报、PPT、UI 卡片网格。
- 每页只保留一个核心图解和一个学习锚点。


## v1.4.0 视觉重构

默认清爽彩色手写学习笔记：黑笔骨架 + 荧光黄关键词 + 浅蓝分区 + 浅绿结论 + 浅粉/红色提醒 + 淡橙路径。不要纯黑白，也不要商业信息图配色。
