# Study Notes Studio 学霸学习笔记

一个把课堂、网课、读书、讲座、播客和复习材料整理成视觉学习笔记的 Codex Skill。

它不只是把文字换成手写体，而是先理解内容、提炼重点，再选择适合的版式，生成清爽、有层次、方便复习的学习笔记图。

## 它能做什么

- 把长内容提炼成 3-5 个核心要点
- 自动找出关键词、知识结构和复习抓手
- 根据定义、流程、对比、分类、公式等内容选择版式
- 把过长内容拆成多页，避免一页塞得太满
- 生成适合回看和复习的中文学习笔记图

## 适合这些内容

- 课堂讲义、老师板书、网课转写
- 读书摘录、论文摘要、文章段落
- 讲座、播客、访谈、会议内容
- 考试复习材料、知识点清单

## 安装方法

1. 下载这个仓库。
2. 把整个 `study-notes-studio` 文件夹放进 Codex 的 `skills` 目录。
3. 重新打开 Codex。

Windows 常用目录：

```text
C:\Users\你的用户名\.codex\skills\study-notes-studio
```

## 使用示例

安装后，可以直接这样说：

```text
用 $study-notes-studio 把这段网课内容整理成一页学霸笔记。
```

也可以说：

```text
帮我先提炼重点，再生成 3 页适合复习的学习笔记图。
```

## 文件说明

- `SKILL.md`：Skill 的核心工作流程和触发说明
- `agents/openai.yaml`：Codex 中显示的名称、简介和默认提示词
- `handbook/`：内容提炼、页面结构、版式、视觉风格和质量检查规则
- `variants/notebook-background/`：仓库保留的真实笔记本背景变体

## 设计目标

让知识更容易理解、记住和复盘。漂亮只是结果，真正重要的是看完有收获。
