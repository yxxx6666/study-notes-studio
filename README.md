# Study Notes Studio 学霸学习笔记

版本：v1.1.0

这是一个用于生成漂亮学习笔记的 Skill。

安装时，把整个 `study-notes-studio` 文件夹放入你的 skills 目录。

核心目标：把课堂、网课、读书、讲座等内容整理成清爽、有重点、能复习的一页视觉学习笔记。

默认背景：干净平铺的单页纸张，不生成笔记本书脊、桌面背景、强透视或实拍样机。

## v1.1.0 更新

- 默认背景改为干净平铺单页纸。
- 强化浅奶白纸纹，避免纯白空白画布。
- 分区优先使用留白、小标签和柔和手绘容器。
- 避免明显竖线、硬切屏、正式表格感。
- 便利贴改为平面轻阴影，避免 3D 样机感。
- 增加版式防模板化规则。
