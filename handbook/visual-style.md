# 视觉风格规范

## 一句话风格

清爽的手写学习笔记：浅色纸张、黑色手写字、少量荧光高亮、清晰分区、一个小图解、克制装饰。

看起来像真实学霸整理出来的复习页。
不是设计师海报。
不是打印讲义。
不是儿童手账。

## 画面比例

固定使用竖版：

- **严格 3:4**
- 推荐按 `1536x2048` 构图理解

除非用户明确要求横版，否则不要使用其他比例。

默认禁止：

- 2:3
- A4 portrait
- 方图
- 横版
- 任意非 3:4 比例

## 背景

默认背景：**全画布学习笔记页**。

整张图片画布本身就是笔记页。不要像一张纸被放在桌面或背景上，也不要像拍摄出来的样机图。

推荐：

- 全画布底色
- 无纸张外框
- 无可见纸张边缘
- 无周围背景
- 浅奶白
- 极浅米白
- 很淡的纸纹或无纸纹
- 干净、不发黄
- 正面平铺视觉
- 无明显透视
- 无强烈阴影

默认禁止：

- 摊开的笔记本
- 书脊
- 独立纸张边缘
- 页面弯曲
- 页面投影
- 桌面背景
- 实拍场景
- 强透视
- 强阴影
- 纸张边缘过度立体
- 纸张放在背景上的样机感
- 纯白刺眼背景
- 纯白空白画布感
- 复古发黄纸
- 牛皮纸
- 咖啡渍
- 复杂背景
- 网格纸过重

只有当用户明确要求“展示图 / 样机 / 笔记本场景 / mockup”时，才可以生成真实笔记本、桌面场景或纸张样机。

## 视觉稳定性

默认成品要像“高级电子学习笔记页”，不是模板化设计稿，也不是纸张样机。

### 画布

- 整张画布就是笔记页。
- 使用浅奶白或柔和米白作为底色。
- 纸纹可非常淡，也可以没有。
- 不要纯白到像空白画布。
- 不要发黄到像旧纸。
- 不要出现纸张外边框。
- 不要出现纸张投影。
- 不要出现周围背景。

### 分区

优先使用：

- 留白
- 小标题标签
- 手绘圆角框
- 轻微底色块
- 自然错落的内容区

避免使用：

- 明显竖线
- 硬切屏左右分割
- 正式表格线
- 商业 PPT 式模块框

### 提醒块

复习提醒默认使用“平面提示色块”，不是现实便利贴。

要求：

- 画在同一画布上
- 完全平面
- 无阴影
- 无纸片边缘
- 无胶带
- 无翘角
- 无立体厚度
- 颜色很淡
- 不要 3D 立体样机感
- 不要成为页面主视觉

只有用户明确要求“便利贴 / sticky note / 手账感”时，才可以生成真实便利贴效果。即便如此，也要保持轻量、克制。

### 版式变化

版式要跟知识结构走。

不要每页都固定成：

```text
标题 + 左右分栏 + 小图解 + 便利贴
```

可以在不同页面中变化：

- 标题位置
- 主体阅读路径
- 图解位置
- 提醒位置
- 分区形状
- 高亮节奏

但变化不能牺牲清晰度。

## 字体感觉

应该是：

- 黑色钢笔或中性笔手写
- 字迹自然
- 行距舒服
- 中文可读
- 不过分潦草

禁止：

- 印刷体讲义
- 花体英文
- 书法字体
- 卡通字体
- 太机械的等宽字体
- 中英文风格混乱

## 颜色

主色：

- 黑色墨水：正文、标题、图解线条
- 荧光黄：核心关键词
- 荧光粉：易错点或对比项
- 浅蓝：少量边注，可选
- 浅黄 / 淡粉：便利贴，可选

颜色原则：

- 一页最多 3 种强调色
- 高亮不要超过 4-6 处
- 颜色用于信息层级，不用于炫技

## 高亮

高亮像真实荧光笔划过。

要求：

- 半透明
- 稍微不规则
- 只覆盖关键词
- 不整行涂满
- 不把正文遮住

## 装饰

可以有：

- 小星星
- 小箭头
- 小灯泡
- 小问号
- 小夹子
- 一张便利贴
- 一小段胶带

但每页只允许少量。
装饰必须服务阅读路径。

禁止：

- 大量贴纸
- 可爱人物占据主体
- 无关插画
- 花边过多
- 阴影很重
- 像手账拼贴海报

## 留白

留白是高级感的来源。

要求：

- 分区之间有呼吸感
- 页边距干净
- 不把文字顶到边缘
- 图解周围留空间
- 页脚提醒不要挤在角落

如果内容太多，拆页，不要压缩字号。

## 整体观感

合格画面应该像：

> 一个成绩很好、审美也好的人，课后认真整理出来的一张干净电子复习笔记页。

默认不是“拍到一本笔记本”或“拍到一张纸”的氛围图，而是可直接保存、打印、发笔记平台或插入文档的成品笔记页。

不合格画面像：

- 老师发的讲义
- PPT 截图
- 密密麻麻的抄写
- 过度装饰的手账
- 摊开的笔记本实拍图
- 纸张放在背景上的样机图
- 带纸张边缘、投影、桌面和强阴影的样机图
- 复习提醒像浮起来的便利贴或 3D 贴纸
- AI 随便生成的漂亮图


## 原创学习锚点图案

每页加入 1 个原创学习锚点图案。

要求：

- 每页 1 个。
- 占画面约 6%–12%。
- 必须贴合本页核心知识点。
- 只能使用普通学习物件、路径符号、记忆符号或抽象结构符号。
- 不能成为角色、吉祥物或主插画。
- 不能挤压文字和图解。
- 不能复用外部案例的构图、动作、轮廓、角色或视觉壳。

推荐：小路标、灯泡、放大镜、钥匙、书签、齿轮、指南针、日历、阶梯、地图针、闹钟、文件夹、记忆抽屉、流程小箭头。

如果图案影响出图，优先简化图案；完整笔记模式下不得为了图案删除知识点。


## v1.4.0 反信息图漂移

默认输出必须更像“手写复习笔记”，而不是信息图、知识海报、PPT 或 UI 卡片页。

禁止：

- 大量圆角卡片
- 每个模块都放图标
- 数据标签铺满页面
- 研究报告式大警告框
- 顶部标题海报化
- 商业信息图模板感

推荐：

- 手写标题
- 自然圈画
- 轻描边分区
- 少量高亮
- 一个核心图解
- 一个原创学习锚点
- 边注式研究提醒

如果页面看起来像展示给别人看的报告，而不是自己复习用的笔记，就要重生。


## v1.4.0 清爽彩色手写笔记风

默认不是极简黑白，而是“黑笔骨架 + 常规学习笔记配色”。

颜色用途：

- 黑色：标题、正文、图解主线。
- 荧光黄：关键词。
- 浅蓝：分区、边注、补充说明。
- 浅绿：正向结论、稳定系统。
- 浅粉 / 红色：易错提醒、不要这样理解。
- 淡橙：路径、箭头、步骤流向。

禁止：

- 纯黑白过度极简。
- 满页彩色模块。
- 高饱和商业信息图配色。
- 每个模块一种大色块。

目标：像真实学生用黑笔、荧光笔和彩色圆珠笔整理出来的复习页。
