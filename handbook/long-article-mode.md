# Long Article Illustration Mode 长文配图模式

长文配图模式不是完整笔记模式。

它的目标是为长文提供章节视觉辅助，而不是把全文所有知识点都放进图片。

## 适合

- 公众号文章
- 长文插图
- 课程讲义配图
- 小红书 / 公众号分节配图
- 每个章节一张视觉总结图

## 不适合

- 要求完整复习
- 要求覆盖所有步骤
- 要求考试背诵
- 要求逐条整理
- 要求不遗漏

只要用户要求完整、不遗漏或覆盖全文，应切换完整笔记模式。

## 执行流程

1. 识别文章结构。
2. 提取 3–5 个章节主旨。
3. 每个章节生成 1 张 3:4 图。
4. 每页控制 40–80 个可见汉字。
5. 图像偏视觉辅助，不做全文承载。
6. 最终报告必须说明：
   - 这是配图。
   - 覆盖主旨。
   - 不代表全文完整。

## 输出页规划格式

```text
长文配图规划：
第 1 页：章节主旨
用于插入：文章第 X 部分后
图上文字：标题 + 3 条短句 + 1 个提醒
不承载：具体数据、引用、细节例子
```

## 禁止

- 不得把“章节配图”说成“完整整理”。
- 不得隐藏已压缩的信息。
- 不得在用户问“内容完整吗”时回答“完整”。
- 不得因为做了 3–5 张图就默认认为覆盖全文。

## 最终报告模板

```text
本次为长文配图，不是完整笔记。
已生成：X 张章节配图。
覆盖：文章主要章节主旨。
未承诺覆盖：全部步骤、数据、例子和细节。
如需完整覆盖，应切换完整笔记模式并执行覆盖矩阵。
```
