# 学习笔记版式配方

## 选择版式的原则

版式由内容决定。
不要为了好看硬套模板。

先判断内容结构：

- 讲定义 → 概念拆解版
- 讲步骤 → 流程路径版
- 讲差异 → 对比辨析版
- 讲分类 → 中心发散版
- 讲公式 → 公式卡片版
- 讲案例 → 场景标注版
- 讲复习 → 错题提醒版

## 版式 1：概念拆解版

适合：定义、术语、理论入门。

页面结构：

```text
顶部：标题 + 一句话定义
中部左侧：这个概念是什么
中部右侧：它解决什么问题
下方：常见误解 / 简单例子
底部：复习提醒
```

视觉特点：

- 关键词高亮在定义句里
- 图解可以画“概念 → 用途 → 例子”

## 版式 2：流程路径版

适合：方法步骤、机制变化、操作流程。

页面结构：

```text
顶部：主题
中部：3-5 个阶段沿着一条手绘路径排列
每个阶段：短标题 + 一句说明
角落：小图解说明关键转折
底部：复习提醒
```

视觉特点：

- 用轻微弯曲的路径，不要正式流程图
- 每一步文字要短

## 版式 3：对比辨析版

适合：A vs B、正确 vs 错误、方法 1 vs 方法 2。

页面结构：

```text
顶部：对比主题
左侧：A 的核心特征
右侧：B 的核心特征
中间：最大差异
下方：什么时候用哪个
底部：复习提醒
```

视觉特点：

- 左右不必完全对称
- 粉色可用于标出易错项
- 不要画正式表格

## 版式 4：中心发散版

适合：分类、框架、一个主题下多个分支。

页面结构：

```text
中心：主题词
四周：3-5 个分支
每个分支：短标题 + 一句解释
边角：例子或提醒
底部：复习提醒
```

视觉特点：

- 分支数量最多 5 个
- 线条要轻，不要像商业脑图模板

## 版式 5：公式卡片版

适合：公式、规律、原则、方法论。

页面结构：

```text
顶部：公式/规律名称
中部：公式本体或核心句
周围：变量解释 / 使用条件 / 常见误区
底部：例题式提醒
```

视觉特点：

- 公式区域要清楚
- 周围标注不要太多
- 易错条件必须突出

## 版式 6：场景标注版

适合：案例、真实场景、实践问题。

页面结构：

```text
顶部：案例主题
中部：一个简单场景图或物件图
旁边：2-4 个标注
下方：从案例提炼出的结论
底部：复习提醒
```

视觉特点：

- 图解占 20%-30%
- 标注短，不要塞长句

## 版式 7：错题提醒版

适合：考试复习、易错点、误区辨析。

页面结构：

```text
顶部：易错主题
左侧：常见错误理解
右侧：正确理解
中部：为什么会错
底部：下次怎么避免
```

视觉特点：

- 对勾/叉号可以用，但要克制
- 粉色用于错误提醒
- 黄色用于正确关键词

## 版式变化规则

同一个版式不要机械重复。
可以变化：

- 标题位置：左上 / 居中 / 顶部偏右
- 图解位置：右上 / 中下 / 左下
- 复习提醒位置：页脚 / 便利贴 / 边注
- 分区形状：圆角框 / 松散区块 / 手绘边线 / 空白分隔

不要变化到失去清晰度。
