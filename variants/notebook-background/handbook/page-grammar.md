# 一页学习笔记的信息语法

## 一页笔记的阅读路径

优秀学习笔记应该有固定阅读路径：

```text
标题 → 一句话总览 → 核心要点 → 图解 → 复习提醒
```

用户不应该在页面里迷路。

## 信息层级

### 1. 标题层

标题是主题，不是场景。

不要写：

- 课堂笔记
- 第一讲
- 读书笔记
- 今日学习

应该写：

- 记忆为什么会遗忘
- 供需关系怎么移动
- RAG 的三步流程
- 批判性思维的核心

要求：

- 不超过 12 个字
- 放在顶部
- 比正文更醒目
- 可以加短下划线或小标签

### 2. 总览层

用一句话告诉读者：这页要解决什么问题。

例：

```text
重点：遗忘不是记性差，而是复习间隔设计错了。
```

总览层可以放在标题下方，也可以放在便利贴上。

### 3. 要点层

要点是主体。

要求：

- 3-5 条
- 每条尽量一行
- 用自己的话写
- 不要逐句搬运
- 不要用密集项目符号

可以使用：

- 小圆圈编号
- 手绘箭头
- 分区标题
- 小图标
- 短横线分隔

但不要做成正式列表。

### 4. 高亮层

高亮词是视觉锚点。

要求：

- 2-4 个关键词
- 只高亮词，不高亮整段
- 黄色为主
- 粉色只用于对比或易错点
- 高亮必须服务复习，不是装饰

### 5. 图解层

图解负责表达关系。

适合画：

- 流程
- 对比
- 分类
- 因果
- 循环
- 组成结构

不适合画：

- 无关插画
- 大面积卡通人物
- 复杂装饰
- 和知识点无关的图案

### 6. 复习提醒层

复习提醒是最后的钩子。

要求：

- 一句话
- 20 字以内
- 最好是结论、易错点或行动提醒

例：

```text
复习时先看关系，不要只背定义。
```

## 页面不能缺什么

一页完整学习笔记至少要有：

- 标题
- 3-5 个要点
- 关键词高亮
- 一个图解或结构关系
- 一句复习提醒

如果只有文字，没有图解，不够像视觉笔记。
如果只有图好看，没有要点，不够像学习笔记。
