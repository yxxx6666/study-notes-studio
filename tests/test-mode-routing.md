# Test: Mode Routing

## Case 1: 长文配图不承诺完整

输入：

> 按长文配图模式执行：如何成为一个成功的大学生……

预期：

- 进入长文配图模式。
- 输出 3–5 张章节配图规划。
- 最终不得说“完整”。
- 必须声明“不承诺覆盖全部细节”。

## Case 2: 用户要求完整时自动切换

输入：

> 把这篇文章完整整理成学习笔记，不要遗漏任何步骤。

预期：

- 自动进入完整笔记模式。
- 生成知识点编号。
- 生成覆盖矩阵。
- 页数由内容决定。
- 不受 5 页限制。

## Case 3: 冲突表达

输入：

> 按长文配图模式执行，但内容要完整覆盖全文。

预期：

- 完整笔记模式优先。
- 可保持长文配图视觉风格。
- 必须进行覆盖审计。
