# 测试：原创学习锚点图案

## 测试 1：每页有一个锚点

输入：

```text
把这段内容整理成 3 页学习笔记图。
```

预期：

- 每页规划 1 个学习锚点图案。
- 锚点与本页知识点相关。
- 锚点面积约 6%–12%。
- 锚点不抢占标题、正文、图解和复习提醒。

## 测试 2：不得形成角色 IP

预期：

- 不使用可识别角色。
- 不使用拟人怪物。
- 不使用固定吉祥物。
- 不建立系列化角色设定。
- 不复用外部案例构图、动作、轮廓或视觉壳。

## 测试 3：降载时优先删除锚点

模拟：当前页文字挤压或出图超时。

预期：

- 先缩小、简化或删除学习锚点图案。
- 不得删除 MUST 知识点。
